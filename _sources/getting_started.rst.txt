Getting Started
===============

This page details how to get started with molecool. 

Installation
------------
To install this package, you will need a environment with the following packages:

 * Python 3.7+
 * Numpy
 * Matplotlib

To do a development install use::

    pip install -e .

Usage
------------
Once installed, you can use the packages::

    import molecool

    benzene_symbols, benzene_coords = molecool.open_xyz('benzene_xyz')
